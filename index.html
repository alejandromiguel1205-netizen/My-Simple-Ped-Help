Welcome

This is the official support portal for My Simple Ped Selector. This tool was designed to simplify the GTA V modding experience, allowing you to manage and rebuild your ped database without manual XML editing.

Download on GTA5-Mods

1. The OpenIV Path

To add your character files correctly, you must navigate to the following path inside OpenIV (make sure Edit Mode is ON):

mods / update / x64 / dlcpacks / mysimpleped / dlc.rpf / x64 / peds.rpf

Option A: Non-Streamed

If you have only 4 files (.ydd, .ytd, .yft, .ymt), drop them directly inside the peds.rpf root.

Option B: Streamed

If your ped is complex, create a new folder inside peds.rpf with the name of the ped and put the files inside.

2. App Configuration

1

Create Entry

Press Ctrl + N. Enter the exact model name (must match the file name in OpenIV). Select gender and set Is Streamed (True if you used a folder, False if not).

2

Rebuild

Click the big REBUILD! button. This compiles your new character into the game's internal database files automatically.

3

Save Workspace

Press Ctrl + G to save your list. If you see a * in the window title, you haven't saved yet!

Fix Errors

Instant Crash on Spawn

Install gameconfig.xml, HeapAdjuster, and Packfile Limit Adjuster. You have exceeded GTA V's memory limits.

Invisible Character

Verify Is Streamed setting. If the files are in a folder in OpenIV, it must be True in the app.
